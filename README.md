# DecodeLabs - Project 2: Automated Quality Inspection

## 📌 Project Overview
This repository contains the deliverables for Project 2 of the Robotics and Automation Internship at DecodeLabs. The project focuses on building a deterministic, real-time Computer Vision system to automate quality inspection on a factory conveyor belt.

## ⚙️ Tools & Technologies Used
* **Language:** Python
* **Library:** OpenCV, NumPy
* **Concepts:** Image Pre-processing (Grayscale, Gaussian Blur, Binarization), Topological Analysis (Contours, Convex Hull, Convexity Defects), PLC Logic Actuation.

## 📂 Repository Contents
* **inspection.py:** The complete Python script for the automated optical inspection pipeline.
* **Project_2_Report.docx:** The detailed technical report explaining the Input-Process-Output (IPO) architecture and threshold calibration.
* **gear_sample.jpg:** Sample dataset used for testing.
* **Output_Screenshots:** Visual proofs demonstrating the system's ability to successfully identify structural defects and display PASS/FAIL verdicts.
